## Introduction

This project is a virtual assistant for the Metaverse Lab, developed by Capgemini. The assistant interacts with users to provide information and perform actions based on user queries. The assistant leverages OpenAI and Azure services for natural language processing and document retrieval. This agent can control an agent in the metaverse like a puppet using the response of the api as a control vector.

## Code Structure

### Main Modules and Libraries

* **Flask:** For creating the web server and API.
* **LangChain:** For managing language models and tools.
* **OpenAI:** For leveraging GPT-4 capabilities.
* **Azure Search:** For document retrieval.

### Key Components

1. **Environment Setup**
   * Loads environment variables using `dotenv`.
2. **Embeddings and Vector Store**
   * Configures Azure Search and OpenAI embeddings.
3. **Functions**
   * `actions()`: Determines actions based on user location and query.
   * `searchDocuments()`: Searches documents using vector embeddings.
4. **API Endpoints**
   * `/chat`: Handles user queries and returns responses with actions.
5. **Tool Classes**
   * `capgeminiDocumentsTool`: Searches documents.
   * `actionTool`: Decides actions based on user inputs.
6. **Prompt Setup**
   * Configures the prompt for the language model to guide its responses.
7. **Response Processing**
   * Formats the AI response into a structured format with actions.
